# Distributed Planning Poker

This will allow for teams to do estimates in a distribute way.

## Getting Started

Clone Git Repo.
Install MariaDB.
Check the flyway setup file to set up flyway and migrate database.
Change the config.py file to match your setup.
Run app.py

### Prerequisites

MariaDB 10.x
