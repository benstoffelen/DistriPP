# Distributed Planning Poker

This will allow for teams to do estimates in a distribute way.

## Getting Started

Clone Git Repo.
Install MariaDB.
Create database "distipp" and run the V1_init.sql file (flywaw coming in future release)
Run app.py

### Prerequisites

MariaDB 10.x
